# DI

For learn about dependency injection (DI) pattern with golang.

## Installation & Usage 
1. Clone the repository:
```git
git clone https://github.com/Coolfreewerx/DI.git
```

2. Install PostgreSQL and PgAdmin4 with Docker:
```bash
docker compose up -d
```

3. Run Program:
```bash
go run main.go 
```
or 
```
air 
```
