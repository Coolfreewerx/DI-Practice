### DI-Pattern

Learn how to use dependency injection and design pattern.

### Installation & Usage 
1. Clone the repository:
```git
git clone https://github.com/Coolfreewerx/DI.git
```

2. Install PostgreSQL with Docker:
```bash
docker compose up -d
```

3. Run Program:
```bash
go run main.go 
```
or 
```
air 
```
